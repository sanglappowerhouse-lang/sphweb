💡 HELPFUL TIPS:
═══════════════════════════════════════════════════════════════════════════════

• Keep all files in the correct folder structure
  (Don't move HTML out of root or CSS out of css/ folder)

• Always refresh browser after making changes
  (Press F5 or Ctrl+Shift+R for hard refresh)

• Use Live Server for automatic refresh
  (Files update in browser automatically as you edit)

• Test on mobile using DevTools
  (F12 → Toggle Device Toolbar → Ctrl+Shift+M)
